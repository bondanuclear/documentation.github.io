Detailed Description

The main Rocket class. It is mostly dedicated to allowing you drive a rocket.

  • RespondToRotateInput Rotates the rocket
  • RespondToThrustInput Accelerates the rocket
  • ApplyThrust Applies thrust to the rocket
  • OnCollisionEnter Tracking the collision
  • StartNextLevelSequence Starts next level sequence
  • StartDeathSequence Starts the death sequence
  • NextLevelSound/DeathSound Sound of getting to the next level and destroying the rocket respectively
  • LoadNextLevel/LoadFirstLevel Loads level

This class is responsible for background sounds, loading levels.
